ES-SIMD Command Guide¶

The following documents are command descriptions for all of the ES-SIMD tools. These pages describe how to use the commands and what their options are. Note that these pages do not describe all of the options available for all tools. To get a complete listing, pass the --help (general options) or --help-hidden (general and debugging options) arguments to the tool you are interested in.

Basic Commands¶

  • s-cg - ES-SIMD code generator
  • s-as - ES-SIMD assembler
  • s-sim - ES-SIMD simulator

Tool Drivers¶

  • s-cc - ES-SIMD compiler driver
  • s-run-sim - ES-SIMD simulation driver
  • s-rtl-gen - ES-SIMD RTL generator
