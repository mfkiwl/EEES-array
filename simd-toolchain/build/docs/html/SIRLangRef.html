SIR Language Reference¶

Abstract¶

The Solver IR (SIR) is a low-level intermediate representation used by ES-SIMD toolchain. It has similar syntax as a typical RISC assembly.

The current version of SIR is type-less. The sematics of an instruction is defined only by its opcode.
